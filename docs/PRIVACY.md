# Privacy Policy & Data Practices

> **Effective Date:** 2025-06-24  
> **Applies to:** NaviEU v0.1.0 and later

NaviEU is built on a radical premise: **navigation software should not require surveillance.**

This document explains what data NaviEU collects, how it is handled, and — more importantly — what we *never* do with your data.

---

## In One Sentence

**NaviEU does not collect, store, or transmit your location data, usage patterns, or personal information by default. Period.**

---

| Practice | NaviEU? |

## What Data Exists (and Where)

All data generated by NaviEU stays on **your device** unless you explicitly choose to export it.

### Data Stored Locally

| Data Type | Purpose | Stored Where | You Control It |
|-----------|---------|-------------|----------------|
| Saved routes | Routes you explicitly save | Device SQLite database | Yes — delete anytime |
| Trail history | Exploration data you choose to record | Device SQLite database | Yes — delete/export |
| Downloaded maps | Offline map tile data | Device file system | Yes — delete anytime |
| App settings | Your preferences (units, theme, voice) | Device preferences | Yes |
| GPX files | Files you import or export | Device file system | Yes — yours to manage |

### Optional Anonymous Analytics

NaviEU includes **opt-in** anonymous analytics to help us understand app stability and usage patterns. This is:

- **Disabled by default** — We ask once on first launch; "No" is the default
- **Fully anonymous** — No device ID, no location, no routes, no personal data
- **Aggregated only** — We see "500 users opened the app today" not "You opened the app"
- **No third parties** — Data goes directly to our self-hosted analytics; no Google, no Mixpanel
- **Revocable** — You can disable analytics anytime in Settings > Privacy

If enabled, we collect:
- App launch count (aggregate)
- Feature usage (aggregate: "map downloads increased 10% this week")
- Crash reports (anonymous stack traces)
- App version and platform type (iOS/Android/Linux)

We **never** collect:
- GPS coordinates or location data
- Route origins or destinations
- POI searches or content
- Device identifiers
- IP addresses (anonymized at collection)

## Technical Guarantees

### No Network Calls During Navigation

When you are navigating, your device makes **zero network requests**. Route calculation, position tracking, map rendering, and voice guidance all happen locally.

### No Cloud Dependency

NaviEU's core functionality does not depend on any cloud service. The app works identically whether you have internet access or not.

### Open Source Verification

NaviEU is fully open source. Anyone can audit our code to verify:
- No hidden network calls
- No data collection mechanisms
- No third-party tracking SDKs
- That the privacy policy matches the implementation

```bash
# Verify no network calls in navigation code
grep -r "http\|dio\|httpClient" lib/navigation/ lib/gps/
# Should return zero results (except for optional map updates)
```

## Map Downloads & Network

The only times NaviEU uses the internet:

| Action | Data Transmitted | Optional? |
|--------|-----------------|-----------|
| Download map regions | Map tile data only | Yes — WiFi-only by default |
| Check for app updates | App version only | Yes — disabled by default |
| Anonymous analytics (opt-in) | Aggregated statistics | Yes — opt-in, revocable |

Map downloads:
- Use HTTPS encryption
- Include no identifying headers or cookies
- Can be configured to WiFi-only
- Can be done on a different network (e.g., home WiFi) before traveling
- Downloaded maps are cached locally indefinitely

## Your Rights (GDPR & Beyond)

Under GDPR and similar privacy frameworks, you have the right to:

1. **Access** — All your data is on your device. You can view and export it anytime.
2. **Deletion** — Delete any or all stored data in Settings > Storage > Clear Data.
3. **Portability** — Export routes and trails as standard GPX files.
4. **Object** — Opt out of optional analytics.
5. **Restriction** — Use NaviEU in airplane mode for complete isolation.

Because NaviEU stores everything locally, exercising these rights does not require contacting us — you have direct control.

## Third-Party Components

NaviEU uses the following third-party open-source components that may have their own data practices:

| Component | Purpose | Network Access | Data Collection |
|-----------|---------|---------------|-----------------|
| OpenStreetMap Data | Map data | None (offline) | None |
| Valhalla | Routing engine | None (embedded) | None |
| MapLibre GL | Map rendering | None (offline) | None |
